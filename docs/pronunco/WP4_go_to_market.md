# WP-4 – Pricing Model

## 1 Tier matrix
| Feature | Free | Pro ($9/mo) |
|---------|------|-------------|
| Local scoring | ✔ | ✔ |
| Azure scoring | ✖ | ✔ |
| Deck export/import | ✔ | ✔ |
| Unlimited custom decks | ⚠ 5 / mo | ✔ |
| Teacher dashboard | ✖ | ✔ |

## 2 Pricing rationale
Azure cost ≈ \$0.0004 / 15-sec clip ⇒ break-even at ~1 k clips / user / mo.
