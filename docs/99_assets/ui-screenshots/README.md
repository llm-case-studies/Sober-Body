# UI Screenshots

This folder contains screenshots for UI development and improvements.
Files here are excluded from git to keep the repository clean.

## Organization:
- `before-{feature}/` - Current state before improvements
- `after-{feature}/` - Improved state after changes
- `issues/` - Screenshots for bug reports or issues
- `ideas/` - Design ideas or inspiration screenshots
- `coach-screen/` - Screenshots related to coach screen improvements

## Usage:
1. Save screenshots here with descriptive names
2. Organize by feature or date
3. Reference in PR descriptions or issues
4. Use for before/after comparisons
